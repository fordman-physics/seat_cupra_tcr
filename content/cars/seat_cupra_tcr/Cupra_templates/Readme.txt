Seat Leon EuroCup 2014 Assetto Corsa Templates Readme:

You need to save these in your personal skin folder

1.
bodytemplate.psd - Carbody Template
File format: DDS - generate mipmap 10 DXT5
Name saving: SLCR_BODY_TEST.dds

2. 
SLCR_WINDOW_TEMPLATE.psd - Exterior Window Template
File format: DDS - generate mipmap 10 DXT5
Name saving: 0TESTWINDOW.dds

3. 
SLCR_PARTS_TEMPLATE.psd - Interior and Exterior plastic and metal paintable parts (boot inside, hood inside, hood carbon mask, front mask, splitter, rear bumper)
File format: DDS - generate mipmap 10 DXT5
Name saving: SLCR_PARTS.dds

4. 
SLCR_FRONT_GRILLE_TEMPLATE.psd - Front Bumper Grille Template
File format: DDS - generate mipmap 6 DXT5
Name saving: SLCR_FGRILLE.dds

I have added a SLCR_FRONT_GRILLE_TEMPLATE2.psd = Its the same as above, but there is now a Text Layer.  Write your own text and save.

5. 
SLCR_RIM_.dds - RIM Template
File format: DDS - generate mipmap 10 DTXT5
Name saving: SLCR_RIM.dds

6.
SLCR_RIM_BLUR.dds - Part 1 of the Rim Blur

7.
SLCR_RIM_BLUR_ALPHA.dds - Part 2 of Rim Blur
You will need to edit both and save in your personal skin folder

8. 
SLCR_BANNER.psd - Front Window Banner Exterior and Interior Template
File format: DDS - generate mipmap 6 DXT1
Name saving: SLCR_BANNER.dds
A part of the black background use the black border on the front and rear window, so leave it black.
